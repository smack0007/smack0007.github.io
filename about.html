Blog

This blog is generated by a custom TypeScript app I wrote just for this purpose. The following open source proects are used:

  • Bootstrap for style.
  • deno to run the compiler.
  • highlight.js for syntax highlighting.
  • html-entities for html sanitization.
  • marked for Markdown conversion.
  • IcoMoon for various images.
  • Sass for SCSS compilation.
