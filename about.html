Me

I'm an American programmer and father currently living in Soest, Germany. I work for Beumer Group. Most of my time I'm living in the C# / .NET world. From time to time I dabble in web programming. With my hobby time I focus on graphics programming using OpenGL thru C#.

Blog

This blog is generated by a custom .NET Core app I wrote just for this purpose. The following open source proects are used:

  • ColorCode-Universal for syntax highlighting.
  • CommonMark.NET for Markdown conversion.
  • Font Awesome for various images.
  • NUglify for css and html minification.
  • SharpScss for SCSS compilation.
